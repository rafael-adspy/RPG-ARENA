🎮 RPG-ARENA

RPG-ARENA é um pequeno jogo de terminal desenvolvido em Python, criado com o objetivo de praticar lógica de programação, estruturas de controle e organização de código.

O projeto simula batalhas simples entre o jogador e a CPU, utilizando um sistema de combate baseado em turnos.

🎯 Objetivo do Projeto

Este projeto foi desenvolvido como parte do meu processo de aprendizado em programação, com foco em conceitos fundamentais como:

Estruturas condicionais (if / elif / else)

Loops (while)

Funções

Manipulação de entrada do usuário

Organização de código

Uso de bibliotecas padrão do Python (time e random)

A ideia é evoluir o projeto gradualmente conforme meu conhecimento em programação aumenta.

⚔️ Sistema de Batalha (Versão Atual)

Na versão atual, o sistema de combate foi simplificado temporariamente para duas ações principais:

[1] Atacar
[2] Curar

Durante o desenvolvimento da primeira versão do sistema de batalha, percebi que a estrutura inicial estava se tornando complexa demais para o estágio atual do projeto.

Para manter o jogo funcional, organizado e evoluindo, optei por simplificar a mecânica de combate nesta etapa.

Essa decisão foi tomada com o objetivo de:

evitar código excessivamente complexo

manter o projeto funcional

continuar evoluindo o aprendizado de forma progressiva
